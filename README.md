# GLAB 386.4.4 — Pandas Aggregate Functions

This lab demonstrates how to use Pandas aggregate functions, including:

- Single aggregate functions (`mean`, `max`, `count`, `std`)
- Multiple aggregate functions using `.agg()`
- Grouping and aggregation using `groupby()`
- Comparing `.agg()` vs direct methods (`.mean()`, `.max()`, etc.)
